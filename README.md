# Barbería Elite - Backend

## La Experiencia del Cliente

Cuando un cliente entra a nuestro mundo, el sistema lo recibe con calidez. Permite explorar los perfiles de nuestros talentosos barberos, cada uno con su especialidad única. Desde cortes clásicos hasta estilos vanguardistas, el backend conecta la demanda con la oferta de manera fluida.

Los turnos se reservan con facilidad, validando horarios comerciales, evitando conflictos y confirmando disponibilidad en tiempo real. Es como tener un asistente personal que conoce los horarios de todos y nunca comete errores.

## El Panel de Control

Para nuestros barberos y administradores, el backend ofrece un panel intuitivo donde pueden gestionar su agenda, confirmar citas y mantener un registro detallado de cada servicio. Las estadísticas diarias permiten optimizar el flujo de trabajo, asegurando que cada día sea productivo y satisfactorio.

## Seguridad y Confianza

En un mundo digital, la confianza es primordial. Este sistema protege la información de nuestros clientes con las mejores prácticas de seguridad, desde autenticación robusta hasta encriptación de datos sensibles.

## Tecnologías Utilizadas

- **Node.js**: El corazón del servidor, rápido y eficiente.
- **Express**: Framework web minimalista para construir APIs robustas.
- **MySQL2**: Base de datos relacional para almacenar toda la información de manera estructurada.
- **bcryptjs**: Para el hashing seguro de contraseñas.
- **cors**: Manejo de políticas de origen cruzado.
- **dotenv**: Gestión de variables de entorno.
- **express-rate-limit**: Limitación de tasa de solicitudes para prevenir abusos.
- **helmet**: Configuración de headers de seguridad HTTP.
- **jsonwebtoken**: Autenticación basada en tokens.
- **multer**: Manejo de uploads de archivos, como fotos de barberos.
